About Weathersite

Weathersite is an app which gives the weather status of different cities. It gives the weather status of different Indian cities which are shown on dashboard. User can also find the weather status of a particular city by entering its name in search option. User can also select a particular city as his favourite city by clicking on it and he can select as many cities as his favourite cities, these favourite cities will be shown under the favourite section of the user.

Only home page will be visible if user is not logged in, to use all the features of this app the user must be logged in.

Some features of this app
